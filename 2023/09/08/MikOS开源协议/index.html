MikOS开源以及使用协议

2024.7.30第二版

MicGan 作

一只悠然自得 审核

MikOS (以下简称本系统) 由 Ark Krin Studio 工作室（以下简称本工作室）MikOS开发组开发，自2.0版对本系统进行开源，使用时请务必遵守本协议

使用者

若您是使用者，请遵守本章协议

注意，若您使用本程序，则默认您同意该协议

1.本系统为基于Python Tkinter 开发的类操作系统的GUI，本系统没有任何收集用户信息的功能，所有用户运行时产生的文件以及隐私文件全部将保存到用户所使用的计算机上，不会上传到任何服务器上（本工作室无法保证第三方应用的程序不会上传文件，本条协议仅限于本工作室开发的系统主程序）

2.您使用本系统后所产生的的任何经济损失以及人身伤害本工作室概不负责

3.若您有任何疑问，请通过邮箱 ArkKrinStudio@outlook.com 联系我们

4.不得私自将本系统用于商业用途，一经发现，本工作室将追究其责任

5.本系统的应用没有任何如收费功能、广告等盈利类功能，如果您发现了这类应用，请通过邮箱 ArkKRinStudio@outlook.com 联系我们，格式如下：版本+应用名称+应用版本+盈利的功能。感谢您的配合!

开发者

若您是开发者，请遵守本章协议

1.本系统基于Python开发，若您想为本系统开发第三应用，请使用Python语言，若您使用其他语言/第三方库，请务必保证向用户说明本应用所使用的语言以及需要安装的运行库 注意: 若您使用其他语言，请考虑您所使用的语言的兼容性以及在本系统上的稳定性

2.所有第三方应用请务必保存在 usr/local/xxx(您的应用程序名) 文件夹中

3.文件命名规则如下(XXX代表您的程序名)：

文件类型 命名方式
主程序 (XXX_)Main
其他文件 XXX 文件名
协议 Licence (可选)
程序说明 Readme

4.若您的程序要上传用户文件，请务必在程序说明中标注所需上传的文件

5.请在Readme文件中说明您开发的应用的用途、所需的运行库以及开发语言

6.不得在应用中加入任何如广告、收费功能的盈利项目，您可以考虑加入非盈利性的广告，但请务必注明，并向本工作室申请，否则我们将追究其责任

7.若您想为本系统的系统文件做出更改或者想加入我们的开发，欢迎在Github上进行PR!。不得非法(包括但不限于植入木马、广告等行为)修改本系统的根文件，否则本工作室将追究其责任

8.不得私自将本系统用于商业用途，一经发现，本工作室将追究其责任
